# Serial Monitor : Realtime time-series data visualizer from a serial port.

## Purpose

This software visualizes time-series data from a serial port in realtime.

## Requirements

- Qt
	Qt5 : http://www.qt.io/
- Qwt support
	Qwt-6.1.1 : http://qwt.sourceforge.net/
